Set Up in 3 Easy Steps

Integrate Sequence directly into your editor to start preparing smarter, not just harder.

1

Install the Extension

Get the Sequence extension from the Visual Studio Code marketplace. It's built exclusively for VS Code to ensure a seamless experience.

Download for Visual Studio Code

Available on macOS and Linux. Windows support is experimental (WSL recommended).

2

Sign In & Talk to the AI

Once installed, open the Sequence sidebar and sign in. Stuck? Just ask! Our AI can guide you through the setup process with voice commands.

3

Configure Your AI Agent

  • Pair with Sequence: Use the chat to get hints, edit code, and understand complex logic seamlessly. Reference files with @filename.
  • Control the AI: Switch between a simple 'Chat' mode for questions and a powerful 'Agent' mode that can read files and suggest edits automatically.
